# dthls

hls v5 impl in c. work with dtcurl


## Installation
- install dtcurl & ffmpeg
- mkdir build & cd build
- make
- sudo make install

## Deps

- [dtcurl](https://github.com/peterfuture/dtcurl)
- [ffmpeg](https://github.com/FFmpeg/FFmpeg)
